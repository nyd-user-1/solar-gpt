# SolarGPT

SolarGPT is a UI replica of QuoteTorch, rebranded for solar roofing installation lead generation. It provides geographic drill-down (state → county → city → ZIP) and a solar installation quote lead form.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.

## Stack

- Next.js 15 (App Router)
- React 19
- Tailwind CSS v4
- TypeScript
- Lucide React icons
- Embla Carousel
